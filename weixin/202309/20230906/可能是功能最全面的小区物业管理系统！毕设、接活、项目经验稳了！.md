想必你已经知道今天我们要介绍的内容，没错，今天要给大家带来的是一个支持一键式自动安装的 HC 小区物业管理系统项目。

## 如何安装

HC 小区管理系统主要有如下三种安装方式:

  * centos7.6 一键式自动安装

    
    
    yum install -y wget && wget http://cdn.homecommunity.cn/hcInstall.sh && sh hcInstall.sh 您的私网IP  
    

注意：您的私网 IP 为服务器内网 ip 如果服务器只有一个私网 ip 时可以省略不填写，如果有多个时需要指定

  * 根据需求自助部署

  * 宝塔方式安装
